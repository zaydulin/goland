"Страница авторизации"
* Форма 
  * Поле: Login
  * Поле: Password
  * Поле: E-Mail
  * Поле: Number phone
* Структура 
  * Подключение шаблона
    * Текущий шаблон
    * Переход на страницу(регистрации)
    * Переход на страницу(сбросить пароль)

  * POST-запрос к базе данных
  * Valid: редерект на страницу профиля

    * При неверном вводе конкретного поля E-Mail -> (GET-запрос) -> ERROR message(Неверный E-Mail)
    * При неверном вводе конкретного поля Номер телефона -> (GET-запрос) -> ERROR message(Неверный Номер телефона)
* База данных 
  * ID
  * Login
  * Password
  * E-Mail
  * Number phone

  * Поле: Отчество
  * Поле: Date

  * Поле: Address
* Форма 2: Изменение учётных данных
  * Поле: Login(текущий)

  * Поле: New_E-Mail(новый)
  * Поле: Number phone(текущий)
  * Поле: New_Number phone(новый)

  * Поле: Текущий пароль(password)
  * Поле: Новый пароль(new_password)
  * Поле: Подтверждение нового пароля(confirmation_new_password)
* "Забыли пароль?" - перенаправление на страницу сборса пароля
* Структура
  * Подключение шаблона
    * Текущий шаблон

    * Переход на страницу(сбросить пароль)

  * Valid: редерект на страницу профиля
  * Invalid: message -> ERROR(filed)
    * При существующем поле Логине -> (GET-запрос) -> ERROR message(Данный логин существует)
    * При существующем поле E-Mail -> (GET-запрос) -> ERROR message(Данный E-Mail существует)
    * При существующем поле Number phone -> (GET-запрос) -> ERROR message(Данный номер телефона существует)
    * При существующем поле Addres -> (GET-запрос) -> ERROR message(Данный Address существует)
* База данных
  * ID
  * Login
  * Password
  * E-Mail
  * Number phone

  * LastName

  * Date
  * Avatar
  * Gender

  * Address

"Страница сброса пароля"
* Форма
  * Поле: Login или E-Mail или Номер телефона
* Структура
  * Подключение шаблона
    * Текущий шаблон
    * Переход на страницу(регистрации)
    * Переход на страницу(авторизации)
  * Вывод формы:
    * Подтверждения смены пароля через E-Mail/SMS
      * Valid:
        * Генерация уникального токена для сброса пароля
        * Сохранение токена в бд со сроком действия
        * Отправка ссылки с токеном на E-Mail
      * Invalid: message -> ERROR(filed)
        * При несуществующем поле Логина -> (GET-запрос) -> ERROR message(Данный логин не существует)
        * При несуществующем поле E-Mail -> (GET-запрос) -> ERROR message(Данный e-mail не существует)
        * При несуществующем поле Номера телефона -> (GET-запрос) -> ERROR message(Данного номера телефона не существует)
        * При неверном вводе подтверждения смены пароля -> (GET-запрос) -> ERROR message(Вы ввели неверный код подтверждения)
  * Страница установки нового пароля(отправленная на E-Mail):
    * Поле: Новый пароль
    * Поле: Подтверждения нового пароля
      * Valid:
        * В случае успешной проверки: пометить токен как использованный
        * Записать в бд новый пароль(password)
      * Invalid:
        * При несовпадении паролей -> ERROR message(Пароли не совпадают) 
        * При совпадении пароля с прошлым -> ERROR message(Пароль не должен быть таким же,как и прошлый)
* База данных
  * ID
  * Login
  * Password
  * E-Mail
  * Number phone
